# AutoStepper Roadmap

## Short term (0-2 months)
- Stabilize CLI parsing and input validation.
- Improve null-safety and file handling.
- Complete Maven migration and CI workflow.
- Add regression tests for BPM and SM generation.

## Mid term (2-6 months)
- Adaptive BPM detection for tempo-shifting tracks.
- Improved pattern generation with stream density controls.
- Batch queue progress reporting and cancellation support.
- Optional metadata/artwork provider API (MusicBrainz/Cover Art Archive).

## Long term (6-12+ months)
- AI-assisted chart generation profiles by style.
- Plugin architecture for custom pattern engines.
- Full UI redesign with chart preview and live diagnostics.
- Cloud/offline hybrid processing pipeline.
